# DoseRight Frontend (React + Vite)

This is a Vite + React + TypeScript app scaffolded to match the provided dashboard image. It expects the backend to run at http://localhost:8080 (Spring Boot) and uses the following endpoints:

- GET /api/medicines
- GET /api/schedule/today
- GET /api/adherence
- GET /api/summary

Install and run:

```cmd
cd d:\Codes\GitHub\DoseRight\frontend
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
